# library
This project is made with javascript and basic css to store the information of books in the form of table like a library which includes information like author,title,no of pages and whether the user has read the book or not.Background color of red and green are applied on the row of the table to indicate whether the user has not read or read the book respectively.

-The project was made to practice the use of objects.
-Table was used instead of grid to practice the use of table.
-New concept of basics of local storage was learnt  

The live demo of the project can be found on:
https://anandb104.github.io/library/
